# retable
Transform table for mobile

##Usage

```html
<table class="retable" data-retable-type="row">
  <tr>
    <th>Title 1</th>
    <th>Title 1</th>
  </tr>
  <tr>
    <td>Content 1</td>
    <td>Content 1</td>
  </tr>
</table>
```

or

```js
retable(selector, type);
```

Type is one of the `['row', 'column', 'separator']`
